# Todo CLI App + monggodb
## Link task: https://gist.github.com/husenEF/4e6cdf2fde384238d0094182248aeee7#file-app-cli-monggodb-md

### Show all todo list
* node app list

### Add new todo item
* node app add "text"

### Update a todo item
* node app update "id" "new text"

### Delete a todo item
* node app delete "id"

### Clear all todo list
* node app clearAll

### Set a todo item to completed
* node app done "id"

### Set a todo item to uncompleted
* node app undone "id"

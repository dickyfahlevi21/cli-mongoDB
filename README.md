## Reference from Panca Putra Pahlawan

### Show all todo list
* node app list

### Add new todo item
* node app add "text"

### Update a todo item
* node app update "id" "new text"

### Delete a todo item
* node app del "id"

### Clear all todo list
* node app clear

### Set a todo item to completed
* node app done "id"

### Set a todo item to uncompleted
* node app undone "id"
